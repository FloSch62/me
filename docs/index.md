## Open source

I'm a maintainer in the [srl-labs](https://github.com/srl-labs) organization — home of
containerlab, which lets you build container-based networking labs with topologies
defined as code. My focus there is the tooling around it.

<div class="grid cards" markdown>

-   :material-lan: **containerlab**

    ---

    Deploy container-based networking labs from declarative topology files —
    the flagship of the srl-labs ecosystem. I'm one of its maintainers.

    [:octicons-arrow-right-24: srl-labs/containerlab](https://github.com/srl-labs/containerlab)

-   :material-microsoft-visual-studio-code: **vscode-containerlab**

    ---

    The official Containerlab extension for VS Code — deploy, visualize and manage
    labs straight from the editor. I'm the lead developer.

    [:octicons-arrow-right-24: srl-labs/vscode-containerlab](https://github.com/srl-labs/vscode-containerlab)

-   :material-monitor: **containerlab-app**

    ---

    Containerlab as a desktop app for Windows, macOS and Linux — built and
    maintained by me.

    [:octicons-arrow-right-24: srl-labs/containerlab-app](https://github.com/srl-labs/containerlab-app)

-   :material-api: **clab-api-server**

    ---

    REST API server for driving containerlab remotely — I'm the lead developer.

    [:octicons-arrow-right-24: srl-labs/clab-api-server](https://github.com/srl-labs/clab-api-server)

-   :material-vector-square: **clab-io-draw**

    ---

    Convert containerlab topologies into draw.io diagrams — and back. I'm the
    lead developer.

    [:octicons-arrow-right-24: srl-labs/clab-io-draw](https://github.com/srl-labs/clab-io-draw)

-   :material-microsoft-windows: **wsl-containerlab**

    ---

    A ready-to-use WSL distribution for network labbing on Windows — one of my
    maintainer hats.

    [:octicons-arrow-right-24: srl-labs/wsl-containerlab](https://github.com/srl-labs/wsl-containerlab)

</div>
